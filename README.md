# VK Download

Android-приложение для просмотра каталога мобильных приложений. Проект представляет собой MVP витрины магазина приложений с локальным хранилищем данных и навигацией между экранами.

## Основные возможности
- **Онбординг**: Стартовый экран приветствия.
- **Каталог приложений**: Список приложений с разделением по категориям.
- **Фильтрация**: Просмотр приложений конкретной категории.
- **Карточка приложения**: Детальное описание и информация о разработчике.
- **Галерея**: Просмотр скриншотов в полноэкранном режиме.

## Стек технологий
На текущем этапе проект использует минимальный набор библиотек для обеспечения базового функционала без внешней сетевой логики:

- **UI**: Jetpack Compose (Material 3) — современный декларативный интерфейс.
- **Navigation**: Navigation Compose — управление переходами между экранами.
- **Data**: Static/Fake Data — данные хранятся локально в виде объектов (без API и БД).
- **Resources**: Статические графические ресурсы (drawable) вместо загрузки изображений из сети.
- **Build System**: Gradle (Kotlin DSL) + Version Catalogs (libs.versions.toml).

## Требования
- Android 6.0 (API 23) и выше
- Android Studio Ladybug (2024.2.1) или новее
- Gradle 8.0+

## Установка и запуск

1. Склонируйте репозиторий:
```bash
git clone https://github.com/ilwwil/vk-download
```

2. Откройте проект в Android Studio.

3. Дождитесь завершения синхронизации Gradle.

4. Соберите и запустите приложение на эмуляторе или физическом устройстве.

## Структура проекта
- `ru.vkdownload.mvp.data.model` — модель данных приложения.
- `ru.vkdownload.mvp.ui.catalog` — список приложений и мок-данные.
- `ru.vkdownload.mvp.navigation` — конфигурация навигации (NavHost).
- `ru.vkdownload.mvp.ui.theme` — темы, цвета и шрифты.
- `ru.vkdownload.mvp.ui.*` — экраны приложения (onboarding, appdetails, categories, screenshots).
