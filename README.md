# RuStore MVP

Учебный Android-проект — упрощённая витрина приложений RuStore.

## Цель
Реализовать MVP магазина приложений:
- онбординг
- витрина приложений
- карточка приложения
- просмотр скриншотов
- (опционально) установка APK

## Стек
- Kotlin
- Jetpack Compose
- Jetpack Navigation
- ViewModel
- Retrofit
- WorkManager

## Минимальные требования
- Android 5.0 (API 21)

## Сборка
```bash
./gradlew assembleDebug
